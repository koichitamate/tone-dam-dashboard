# 利根川水系9ダム貯水率ダッシュボード

## プロジェクト概要

利根川水系の主要9ダムの貯水率を、公開データをもとに一覧表示するダッシュボードアプリです。
防災判断ではなく、公開情報を見やすく可視化することを目的とした**学習用アプリ**として開発します。

## 初期開発方針

1. **データ取得の検証を先行する**
   - 国土交通省などが公開する貯水率データを取得できるか確認する
   - APIの有無、スクレイピングの可否、データ形式（JSON / CSV / HTML）を調べる

2. **Reactでダッシュボード画面を構築する**
   - 検証後、取得したデータをReactで可視化するUI を実装する
   - 貯水率の一覧表示・グラフ表示を基本機能とする

3. **学習用アプリとして設計する**
   - 防災判断や避難指示には使用しない
   - 公式の公開情報をわかりやすく表示することを目的とする

---

## 初期フェーズでは実装しないこと

以下の機能はスコープ外とし、初期フェーズでは一切実装しない。

- ログイン・認証
- プッシュ通知・アラート通知
- 長期履歴データの保存
- 避難判断・防災判定ロジック

---

## 開発ステップ

### Step 1 — データ取得の検証
- 公式公開データ（国土交通省 水文水質データベース等）から貯水率を取得できるか調査
- データ形式・更新頻度・利用条件を確認
- サンプルスクリプトで取得テストを実施

### Step 2 — バックエンド（データ取得層）の構築
- 定期的にデータを取得・整形するスクリプトまたはAPIエンドポイントを実装
- 9ダム分のデータを統一フォーマットに変換する

### Step 3 — フロントエンドの構築
- Reactでダッシュボード画面を実装
- 9ダムの貯水率を一覧表示
- グラフ（棒グラフ・ゲージ等）での可視化

### Step 4 — 統合・動作確認
- バックエンドとフロントエンドを接続
- 実データでの動作確認・表示調整

### Step 5 — 公開・改善（任意）
- Vercel等への静的ホスティング
- UIの改善、追加情報の表示検討

---

## 注意事項

- 本アプリは学習目的で作成しています
- 表示する貯水率データは公式機関の公開情報に基づきますが、最新性・正確性を保証するものではありません
- 防災・避難判断には必ず公式の情報をご確認ください
